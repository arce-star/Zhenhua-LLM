# Zhenhua-LLM: 八月长安文风复刻大模型

本项目旨在通过大模型微调技术，复刻著名作家八月长安（“振华三部曲”）的细腻文风。

## 🚀 模型特性
- **细腻心理描写**：深入学习了洛枳、耿耿、余周周等角色的内心独白逻辑。
- **长文生成优化**：针对“挤牙膏”问题进行了长文本训练，输出更流畅。
- **振华宇宙记忆**：对红榜、校服、天台、单车等经典意象有极高敏感度。
